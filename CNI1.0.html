Introduction

Le Changement de Carte Nationale d'Identité (CNI) est un processus par lequel les citoyens peuvent mettre à jour leurs informations personnelles sur leur carte d'identité. Cela peut être nécessaire en cas de changement d'adresse, de nom, etc.

Processus

Le processus de changement de CNI implique les étapes suivantes :

  1. Remplir le formulaire de demande de changement de CNI.
  2. Préparer les documents justificatifs nécessaires.
  3. Se rendre au bureau des services d'identité pour soumettre la demande et les documents.
  4. Attendre le traitement de la demande.
  5. Recevoir la nouvelle carte d'identité mise à jour.

Conditions requises

Pour effectuer un changement de CNI, les conditions suivantes doivent être remplies :

  • Être citoyen du pays.
  • Avoir atteint l'âge de majorité.
  • Fournir les documents originaux pour prouver les modifications (par exemple, un certificat de mariage pour un changement de nom).
